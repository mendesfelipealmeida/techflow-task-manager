# TechFlow Task Manager

## Objetivo do Projeto
Este projeto simula o desenvolvimento de um sistema de gerenciamento de tarefas usando práticas de Engenharia de Software e metodologias ágeis.

O objetivo é permitir que o usuário:
- Cadastre tarefas
- Edite tarefas
- Exclua tarefas

Funcionalidade implementada: **CRUD de tarefas** (Create, Read, Update, Delete).

---

## Escopo Inicial
- Criar um sistema simples com apenas o botão de adicionar tarefas.

---

## Mudança de Escopo (Gestão de Mudanças)
Durante o desenvolvimento, o cliente solicitou uma alteração no sistema:

> **"Além de adicionar tarefas, queremos também editar e excluir."**

✅ Escopo atualizado no quadro Kanban  
✅ Código alterado e commitado no repositório

Essa alteração foi registrada no projeto como gestão de mudanças.

---

## Metodologia Utilizada
Foi utilizada uma adaptação de **Scrum + Kanban**:

- Organização das atividades em *Issues*
- Planejamento visual no quadro **Projects (Kanban)**
- Commits realizados conforme avanço das tarefas

Colunas do Kanban:
- A Fazer
- Em Progresso
- Concluído

---
